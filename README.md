# Calculus API

## How to setup
```sh
$ git clone git@github.com:barbarafarias/calculus.git
$ cd calculus
```

## How to run

### With Docker
```sh
$ docker-compose up
$ Go to http://localhost:3000/calculus?query=Mis4KjktKDEwLzQp
```

### Without Docker
```sh
$ npm install
$ npm start
$ Go to http://localhost:3000/calculus?query=Mis4KjktKDEwLzQp
```
